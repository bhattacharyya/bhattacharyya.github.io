Exploring Data

Tools for the trade

Tue 03 November 2015

OK, here is the list of tools that you might find immensely useful in getting started with data exploration. Some of these things are very basic and you probably have years of experience with those already. Some others are quite new and gaining in popularity fast. Remember though that these are mostly suggestions and if you are more comfortable with a similar package in the same category, definitely stick to that. Also, this list is far from being exhaustive. I will add more things here as I expand my toolset or find a better way of doing the same things.

Python

If you are just starting out with python, starting with python 3 (3.4 or higher) might make more sense but the choice is not very obvious. Python's biggest appeal is the tremendous community support and most of the support available out there is for version 2.7. Python 3 is the future though and there is no getting around that fact. So, one strategy could be to get used to the "pythonic way" using version 2.7 and eventually move to version 3+ when the basic language constructs are firmly set in your brain. Ultimately, choosing either version for now is gonna be all right.

IPython

Remember the days when you thought gedit or notepad was enough. Learning vim or emacs was unnecessary. Well, now you know better. You do right ?? Similarly, the python shell is pretty nice but IPython is something you absolutely need. Running ".py" scripts is fine and sometimes even necessary but the power of using Ipython as a shell is the visualization features that go far beyond plain text. It is particularly productive in combination with ...

pandas

Get it and get used to it. You will thank me later.

matplotlib

Yes you absolutely need it. You can use more advanced stuff like seaborn but try not to complicate things at the beginning.

SQL/SQLite

Unless you are dealing with relatively small datasets, database will make your life much easier. You can pick any database engine but I personally prefer SQLite. In my limited experience, it works pretty well and is easy to learn.

Lots of public data

There are many sites that provide free access to tons of data. Apart from the link above there is Kaggle or reddit data sites. You might even want to start following data science groups on social platforms like twitter. Not only will you find wonderful insights from diverse datasets but also understand how the experts approach the whole process of data analysis.

I recently got hold of the entire dataset of public reddit comments (~1TB). Kaggle even hosted a competition where they provided the data for month of May 2015 in database format.

I will post some quick analysis soon. It probably won't be any pathbreaking insight but it should get me started with data exploration started. Looking forward to working with actual data finally !!
